# industrial_robot_platform
1. Operation system - Ubuntu 22.04 Desktop https://releases.ubuntu.com/22.04/
2. USE UTF-8:
      `sudo apt update && sudo apt install locales/n  
      sudo locale-gen en_US en_US.UTF-8  
      sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8  
      export LANG=en_US.UTF-8`  
3. Install ROS2 Humble - https://docs.ros.org/en/humble/Installation.html
